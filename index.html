Getting Started

Launching Synthara

Synthara is a terminal-based application for interacting with language models from your command line.

cd build
./synthara

Navigation

  • ↑/↓ Move between options
  • Enter Select option
  • F10 Exit application

Main Menu

  • Select Provider - Google
  • Select Model - Choose language model
  • Set API Key - Enter API credentials
  • Set API Token - Enter authentication token
  • Configure Port - Set connection port (default: 8080)
  • Start Chat - Begin conversation
  • Set System Message - Configure AI instructions
  • Customize Theme - Select theme (Light/Dark/System) or customize colors
  • Company Info - About LibertyMind
  • Exit - Close application

Setting Up

  1. Select Provider

    Choose "Google" from the provider menu.

  2. Set API Key

    Enter your Google API key.

    API key and token required for access
  3. Configure Port

    Default port is 8080. Change in settings if needed.

  4. Select Model

    Choose from available Google models:

    • Gemini 1.5 Pro - Long context model
    • Gemini 1.5 Flash - Faster, efficient model
    • Gemini 2.0 Flash-Lite - Cost-effective model

Using the Chat

Chat Session

After setup, select "Start Chat" from the main menu to begin.

Controls

  • Enter Send message
  • Esc Return to menu
  • M Export to Markdown
  • Backspace Delete text

System Message

Configure AI behavior with a system message:

  1. Select "Set System Message" from menu
  2. Enter instructions
  3. Press Enter to save
Example: "You are a helpful assistant that explains technical concepts simply."

Markdown Export Feature

Export your chat history as a Markdown file with a single keystroke.

Export Process

  1. During chat

    Press M to open export dialog

  2. Save location

    Enter file path or use default

  3. Complete

    Press Enter to save

Press Esc to cancel export

Exported Content

  • Title and timestamp
  • System instructions
  • Complete conversation
  • Synthara footer
